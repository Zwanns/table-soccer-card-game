# Отчет по разработке

## Проект

Браузерная карточная футбольная игра `Total Soccer: Mundial`.

Основная спецификация: `GAME_SPEC.md`.

Текущая версия приложения: `0.7a`.

## Текущий статус

Завершены этапы 0, 1, 2, 3, 4 и 5 из `GAME_SPEC.md`.

Этап 6 закрыт по актуальной концепции проекта: интерфейс матча уже существенно переработан, добавлены анимации, табло, выбор сборных, флаги, звуки, шкала превосходства, финальная статистика, улучшенные правила взаимодействия и подтверждение выхода из незавершенной партии.

Некоторые исходные требования этапа 6 больше неактуальны:

- подсветка текущей линии атаки не добавляется намеренно, чтобы игрок сам оценивал карты;
- история последних 5 событий скрыта из интерфейса, но журнал событий остается во внутреннем состоянии;
- настройка имен игроков заменена выбором национальных сборных.

Следующий чистый этап по `GAME_SPEC.md`: этап 7, сохранение и восстановление локальной партии через `localStorage`.

## Ключевые правила текущей версии

Атака идет по линиям:

```text
полузащита -> защита -> вратарь
```

Игрок кликает по карте актуальной линии соперника.

Если карту можно побить:

```text
карта соперника удаляется
карты попадают в attackBank
атака продолжается
```

Если карту нельзя побить:

```text
карта соперника остается на поле
атака завершается потерей мяча
ход переходит сопернику
```

Для `GK` действуют специальные исходы:

```text
гол
штанга
сэйв
```

Для вратарей от `3` до `10` гол возможен только картой строго выше номиналом. Равный номинал дает штангу.
