# Total Soccer: Mundial

Версия: 1.3.0

Браузерная карточная футбольная игра для локальных матчей HUMAN vs HUMAN, HUMAN vs AI, AI vs HUMAN и AI vs AI. Проект совмещает футбольный матч, классические игральные карты, сборные, турнирные режимы, серии пенальти, статистику матча и локальное редактирование составов.

## Возможности

- Матч на Phaser с футбольным полем, счетом, журналом событий, колодами, картами и индикатором преимущества.
- Локальная игра для людей и встроенного AI: красная и черная колоды, а также отдельные карты голкиперов.
- AI-режим для быстрого матча и визуальных матчей турнира без уровней сложности.
- Строгие правила подключенных полузащитников: равный rank не пробивается, специальные правила сохраняются.
- 64 национальные сборные с локальными флагами, цветами формы и обложками колод.
- Экраны выбора команд, выбора составов и редактор составов.
- Турнирные режимы с групповой стадией, плей-офф, симуляцией матчей и сериями пенальти.
- Финальная статистика матча: авторы голов, владение, попадания в штангу, сейвы голкиперов и реализация моментов.
- Автоматические тесты для правил карт, игрового движка, составов, командных ассетов и турнирной логики.

## Технологии

- TypeScript
- Phaser
- Vite
- Vitest
- HTML и CSS

## Быстрый старт

Установить зависимости:

```bash
npm install
```

Запустить dev-сервер:

```bash
npm run dev
```

Запустить тесты:

```bash
npm run test
```

Собрать production-версию:

```bash
npm run build
```

Готовая сборка создается в папке `dist/`.

## Структура проекта

```text
src/
  ai/           Эвристический AI, типы контроллеров и таймерный контроллер AI-хода
  cards/        Модели карт, генерация колод и карточные правила
  data/         Сборные, составы, формы и валидация данных
  game/         Состояние матча, игровой движок, правила поля и статистика
  scenes/       Phaser-сцены меню, настроек, матча, результатов и турнира
  services/     Локальное хранение составов
  tournament/   Турнирный движок, сетка, таблицы и серии пенальти
  ui/           Переиспользуемые Phaser UI-компоненты
  tests/        Тесты Vitest
public/
  covers/       Обложки национальных колод
  flags/        Локальные SVG-флаги
  kits/         Публичные ассеты формы
  menu/         Ассеты главного меню
cards/          Изображения игральных карт
Sounds/         Звуковые эффекты матча
Fonts/          Локальные шрифты
```

## Команды

| Команда | Описание |
| --- | --- |
| `npm run dev` | Запуск Vite dev-сервера |
| `npm run test` | Однократный запуск тестов Vitest |
| `npm run test:watch` | Запуск Vitest в watch-режиме |
| `npm run build` | Type-check и production-сборка |

## Примечания

- Игра рассчитана на запуск в браузере.
- Основные визуальные ассеты лежат локально, без зависимости от внешнего хостинга изображений.
- Пакет помечен как private и используется как приложение, а не как npm-библиотека.
